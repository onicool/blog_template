## インストール
    https://git-scm.com/
## 初期設定～PUSHまで
開発中は３から５を繰り返す。

1. リポジトリ作成（ローカルでgit管理を開始する）
    ```
    git init
    ```
2. リモートリポジトリとの紐づけ（オンラインのgithubとの紐づけ）
    ```
    git remote add https://XXXXX
    ```
3. ファイル追加（リポジトリに入れるファイルを選ぶ。まだローカル）
    ```（ファイル全部の場合）
    git add .
    ```
4. コミット（リポジトリに記録。まだローカル）
    ```
    git commit -m "first commit"
    ```
5. プッシュ（リモートリポジトリにアップロード）
    ```
    git push -u origin main
    ```
    git branch -M main
    git remote add origin
