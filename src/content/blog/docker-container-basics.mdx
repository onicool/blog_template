Dockerは、アプリケーションの開発・デプロイを効率化するコンテナ技術です。

## Dockerfileの基本

```dockerfile
FROM node:18
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
CMD ["npm", "start"]
```
